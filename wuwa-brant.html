Brant

Overview

Quick starter guide for Brant.

Build

Echoes: Main set matching element; sub-stats CRIT > ATK% > Energy/Res.
Weapons: Signature if possible; craftable alternative is fine.

Teams

  • Pair with compatible Resonators for strong Intro/Outro buffs.
  • Maintain Liberation cycles.

Rotation

Intro → skill combos → dodge cancels → Liberation on buff windows.

Notes

Prioritize CRIT and comfort Energy; practice animation cancels.
